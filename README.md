## Simple Weather App
Repo for the Simple Weather App tutorial on [Medium](https://medium.com/p/204bba8a1667/edit)

## Install
``` 
npm install
```
## Add API Key
Edit the `apiKey` variable in the `App.js` file with your API key from [OpenWeatherMap](https://openweathermap.org/api)
## Run
```
npm start
```
